
# Netflix Dashboard - Power BI

## Project Overview
This project presents an interactive Power BI dashboard analyzing Netflix titles dataset.

## Dashboard Features
• Movies vs TV Shows distribution  
• Netflix content growth by year  
• Top countries producing Netflix content  
• Content distribution by rating  
• Interactive slicer filters  

## Tools Used
Power BI, Data Visualization, Data Cleaning

## Dataset
Netflix Titles Dataset (CSV)
